# DocFinder

DocFinder is a modern, responsive web application for finding medical specialists near you. Built with Vite + React.

## Features

- **Search Doctors**: Find specialists by name, condition, or category (Dental, Vision, etc.).
- **Geolocation**: Simulate finding doctors near your location.
- **Book Appointments**: Integrated with Google Calendar for easy scheduling.
- **Premium Design**: Clean, glassmorphism-inspired UI.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Tech Stack

- React
- Vite
- Vanilla CSS (Premium Custom Variables)
